## 🧠 Sobre o jogo
Este é um **jogo de puzzle baseado em criação de senha**.

O objetivo é criar **uma única senha** que satisfaça **dezenas de regras diferentes**.  
Algumas regras são simples, outras são completamente absurdas.

Você terá que usar:

- números
- emojis
- palavras específicas
- símbolos
- referências culturais
- matemática
- lógica
- e muita paciência 😈

No total existem **65 regras**.

---

## 🕹️ Como jogar

1. Abra o link do jogo no navegador.
2. Digite uma senha no campo principal.
3. As regras aparecerão na tela.
4. Cada regra fica **verde quando é satisfeita**.
5. Continue modificando a senha até completar **todas as regras**.

Quando todas forem cumpridas:

🎉 **Você vence o jogo.**

---

## ⚙️ Mecânicas do jogo

### Sistema de Regras
Cada regra possui:

- um texto explicando o desafio
- uma função que verifica se a regra foi cumprida

O jogo avalia todas as regras continuamente enquanto o jogador digita.

---

### Revelação Progressiva
As regras não aparecem todas de uma vez.

Conforme você resolve algumas, **novas regras aparecem**, deixando o desafio cada vez mais caótico.

---

### Chips de Caracteres
O jogo possui um painel com **caracteres clicáveis**, como:

- π
- ∞
- √
- 🇳🇱
- 🇧🇷
- 🔥
- ❤️
- 🎶

Isso facilita inserir símbolos que seriam difíceis de digitar manualmente.

---

### Contagem de Grafemas
O jogo usa **contagem real de caracteres visuais**, incluindo emojis e bandeiras.

Isso significa que:

- `🔥` conta como **1 caractere**
- `🇧🇷` conta como **1 caractere**

Mesmo que internamente sejam vários códigos Unicode.

---

## 🎯 Objetivo

Criar **uma senha única** que satisfaça **todas as regras do jogo**.

Algumas exigências incluem coisas como:

- conter certos emojis
- conter números específicos
- conter palavras específicas
- terminar com `!`
- ter contagens específicas de caracteres ou vogais

Tudo ao mesmo tempo.

Boa sorte 🙂
